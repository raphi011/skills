# Step 5: Create/Update PR

Only after ALL checks pass.

## Check for Existing PR

```bash
gh pr view --json number 2>/dev/null
```

## If No PR Exists

1. Push branch if needed: `git push -u origin HEAD`
2. Check for PR template: `ls .github/pull_request_template.md .github/PULL_REQUEST_TEMPLATE.md 2>/dev/null`
3. Create PR using template if available: `gh pr create --fill`
4. Otherwise create with summary format from `templates/pr-summary.md`

## If PR Exists

1. Push any new commits
2. Update PR description if needed

## Summary

After PR creation/update, generate summary per `templates/pr-summary.md`.
